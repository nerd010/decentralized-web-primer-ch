# 简介

## 安装和初始化 IPFS

这些教程是使用 go-ipfs 0.4.4 的版本进行测试的。请使用在 github 上已经测试过的文件版本。

## 要求

需要有一些命令行的基础

## 学习目标

这些教程将教你如何

- 安装 IPFS
- 初始化一个 IPFS 的仓库
- 定位 IPFS 存储在你本地的 IPFS 仓库的内容

## 重点

- IPFS 仓库

## 教程

1. 下载并安装 IPFS
2. 初始化你的 IPFS 仓库

## 下一步
